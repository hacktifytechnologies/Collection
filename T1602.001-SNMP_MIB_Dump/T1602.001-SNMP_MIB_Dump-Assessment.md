# T1602.001 – SNMP MIB Dump | Assessment

## MCQ 1
SNMPv2c with community string "public" is a critical vulnerability because:
A) SNMP is always encrypted  B) The community string acts as a password — "public" is the default and allows read-only access to the entire MIB tree  ✅
C) SNMPv2 requires authentication  D) SNMP only exposes non-sensitive data

## MCQ 2
`snmpwalk -v2c -c community host` collects:
A) Only network interface data  B) All OID values in the MIB tree including system info, network config, process lists, and custom extensions  ✅
C) Only version information  D) Routing tables only

## MCQ 3
SNMP `extend` directive allows attackers who control snmpd.conf to:
A) Extend the SNMP protocol version  B) Execute arbitrary commands and expose their output as SNMP OIDs  ✅
C) Add new community strings  D) Encrypt SNMP traffic

## Fill 1
SNMP community string used for read-only access by default (most dangerous):
**Answer:** `public`

## Fill 2
SNMPv3 security model that provides authentication and encryption (mitigation):
**Answer:** `authPriv` (authentication + privacy/encryption)
