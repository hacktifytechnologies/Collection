# T1056.001 – Keylogging | Assessment

## MCQ 1
`/proc/bus/input/events` is used by keyloggers to:
A) Read process memory  B) Access raw keyboard input events at the kernel level without needing a windowing system  ✅
C) Read clipboard data  D) Monitor network traffic

## MCQ 2
`pynput` library keyloggers are detectable because:
A) pynput is malicious software  B) The process accesses X11 keyboard hooks or /dev/input devices — unusual for most server processes  ✅
C) pynput requires root  D) pynput only works on Windows

## MCQ 3
Keylogging is classified as T1056.001 (Input Capture) because:
A) It captures screenshots  B) It intercepts user input (keystrokes) to collect credentials and sensitive information  ✅
C) It reads memory  D) It captures network packets

## Fill 1
Linux device file that provides raw keyboard input events:
**Answer:** `/dev/input/event0` (or similar eventN)

## Fill 2
auditd rule to detect reads of keyboard input devices:
**Answer:** `-w /dev/input -p r -k keylogger_detect`
